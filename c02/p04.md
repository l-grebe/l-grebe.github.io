# 进入插入模式
从普通模式切换到插入模式

```shell
i 在当前字符前插入字符
a 在当前字符后插入字符

I 在当前行行首插入字符
A 在当前行行末插入字符

o 在当前行下插入新的一行
O 在当前行上插入新的一行

r 替换光标所在的字符，只替换一次
R 一直替换光标所在的字符，一直到按下ESC
```

 从普通模式删除指定内容，然后切换到插入模式

```shell
cw  删除当前光标到该单词末尾的部分内容，并进入插入模式
cb  删除当前光标前一字符到该字符所属单词开头的部分内容，并进入插入模式

c^ 删除该行游标前面所有字符(不包括开头的空格)，并进入插入模式
c0 删除改行游标前面所有字符，并进入插入模式
c$ 删除游标所在处，到该行的最后一个字符，并进入插入模式

ciw 仅删除当前光标所在单词，并进入插入模式（常用）
caw 删除当前光标所在单词以及两次的空白字符，并进入插入模式

c3iw 仅删除当前光标所在的3个单词，并进入插入模式（常用）

cis 删除当前句子，并进入插入模式
cip 删除当前段落，并进入插入模式
```
