# go 字符串

###     

- byte, rune, string, []byte
-

样例1：观察字节和rune的区别

```go
package main

import "fmt"

func main() {
	s := "hello, 世界"
	for i, r := range s {
		fmt.Printf("%d\t%T\t%q\t%d\n", i, r, r, r)
	}
	fmt.Println()
	for i := 0; i < len(s); i++ {
		fmt.Printf("%d\t%T\t%q\t%d\n", i, s[i], s[i], s[i])
	}
}
```
